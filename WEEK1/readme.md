# ReactJS - WEEK1

- A small React app for fetching, filtering users, light/dark ... implementation using Vite project and Tailwind CSS
- Objectives : Learning, creativity, code quality, anti-ai use (except the implementations of complicated tasks or getting stuck!)

## Team Members

- Saad Ouafir
- Abdelaziz Khouda

## Tech Stack

- React
- Vite
- Tailwind CSS

Key folders/files:

- `src/components` — Reusable UI components (e.g., `NavBar.jsx`, `UsersCard.jsx`).
- `src/hooks` — Custom hooks (`useFetch.js`, `useFilterData.js`, `useLocalStorage.js`, `useSwitchTheme.js`, `useTimer.js`, `useToggle.js`).
- `src/main.jsx` — Application entry point.
- `index.html`, `vite.config.js` — Vite configuration and HTML template.

## Prerequisites

- Node.js (recommended 16.x or later)
- npm (or a compatible package manager)

## Setup

1. Install dependencies

```bash
npm install
```

2. Run development server

```bash
npm run dev
```
